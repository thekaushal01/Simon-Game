# Simon Game

A simple web game inspired by the classic Simon memory challenge.

## What it does
- Starts when you press any key.
- Shows a growing color sequence, one step per level.
- You repeat the sequence by clicking the colored buttons.
- The game ends when you click a wrong color.

## Project files
- `index.html` - page structure
- `styles.css` - game styles
- `app.js` - game logic

## How to run
1. Open `index.html` in your browser.
2. Press any key to start.
3. Follow and repeat the sequence.

## Controls
- Keyboard: any key to begin or restart after game over.
- Mouse: click the color buttons to enter your sequence.

## Notes
- This is a beginner-friendly front-end project built with plain HTML, CSS, and JavaScript.
